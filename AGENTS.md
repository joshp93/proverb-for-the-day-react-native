# Proverb for the day

This is an expo project which is primarily targetted at mobile users. It is quite simple, it just calls out to an api to get the proverb for the day and show it within the app.

When making changes, bear the following in mind:

- You cannot modify generated files, since these are generated by expo and all changes will be overridden upon build. examples include the android folder.
- All changes need to be inline with expos guidance and documentation
- Unless explicitly asked, avoid building or running the project, just stick to making changes. If it is necessary to build or run as part of a task, please ask first.
