# Proverb for the day

This is an expo project which is primarily targetted at mobile users. It is quite simple, it just calls out to an api to get the proverb for the day and show it within the app.

## Ultimate Goals

- ✅ **Widget** - Have a widget which can be shown on phone home screens to display the proverb for the day (which updates daily automatically)
  - Android: Fully implemented with daily 5:05 AM updates
  - iOS: Base components ready, native WidgetKit implementation needed
  - See `WIDGET_IMPLEMENTATION.md` for complete details

- Send push notifications to users once per day to encourage people to meditate on the proverb for the day
- Have a notes / comments system, allowing people to capture their thoughts about a proverb. These are public and can be seen by other users when they are looking at the "meditations" section for a proverb. They can also be made private by choice.

The back end is a separate repo and will be AWS, including some serverless lambda functions. Any changes beyond this apps direct functionality will need to be added to the API.

## Development Guidelines

When making changes, bear the following in mind:

- This project uses pnpm and typescript
- Unit tests are important
- You cannot modify generated files, since these are generated by expo and all changes will be overridden upon build (examples include the android folder - EXCEPT native code files like ProverbWidgetProvider.kt which are in src/main/java)
- All changes need to be inline with expo's guidance and documentation
- Unless explicitly asked, avoid building or running the project, just stick to making changes. If it is necessary to build or run as part of a task, please ask first
- Please remember to optimise the imports

## Widget Implementation

The widget system has been implemented and includes:

- **Storage Service** (`app/_services/widget-storage.ts`) - Persists proverb data
- **Background Task Service** (`app/_services/widget-background-task.ts`) - Schedules 5:05 AM daily updates
- **Widget Component** (`app/_components/proverb-widget.tsx`) - React Native widget UI
- **Android AppWidget** (`android/app/src/main/java/com/proverbfortheday/app/ProverbWidgetProvider.kt`) - Native implementation
- **Comprehensive Tests** - 40 passing tests covering all widget functionality

For implementation details, troubleshooting, and future iOS WidgetKit development, see `WIDGET_IMPLEMENTATION.md`.
